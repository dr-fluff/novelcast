# 📖 NovelCast

A text-based "podcast" experience for web stories.

NovelCast lets you download, organize, and read web stories locally — with automatic updates and a clean reading experience. Think of it like a podcast app, but for written content instead of audio.

## Features
* 📥 Download web stories for offline reading
* 📚 Store and manage stories in a local database
* 🔄 Automatic updates for ongoing stories
* 🧭 Organized reading experience (chapters, progress tracking, etc.)
* ⚡ Fast and simple interface


## 🧠 Concept

NovelCast is built around a simple idea:

Web novels should be as easy to follow as podcasts.

Instead of manually checking websites for updates, NovelCast keeps track of your stories, downloads new chapters, and lets you read everything in one place.

## 📦 Installation
```
git clone https://github.com/dr-fluff/novelcast.git
```
